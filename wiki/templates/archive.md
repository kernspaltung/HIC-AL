- Featured Contents
   1-3 Featured Contents
- Chronological Content List
- Each Content Item has:
   - Title
   - Publication Date
   - Short Description
   - Image
   - Categories
   - Content Type indicator
   - "View More" Link
- Navigation
   - Content Type
   - Theme Filtering
   - Date Filtering
